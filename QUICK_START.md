# ⚡ Quick Start : Nouvelle Fonctionnalité Calendrier

## 🎯 En Bref

Cliquez sur une date dans le calendrier → La date est automatiquement pré-remplie → Ajustez les heures → Créez !

## 🚀 Utilisation

### 1. Ouvrir le Calendrier
```
Dashboard → Cron Calendar
```

### 2. Cliquer sur une Date
Exemple : Cliquez sur "1er mai"

### 3. Le Formulaire s'Ouvre Pré-rempli
```
✅ Date : 1 mai 2026
✅ Heure début : 08:00
✅ Heure fin : 20:00
✅ Cron généré : 0 8 1 5 * / 0 20 1 5 *
```

### 4. Ajuster (Optionnel)
Modifiez les heures si nécessaire

### 5. Créer
Cliquez sur "Create" → Terminé !

## 💡 Exemple Concret

**Besoin** : Scaler une app le 15 juin de 9h à 18h

**Actions** :
1. Clic sur "15 juin"
2. Modifier heure : 09:00 → 18:00
3. Clic "Create"

**Temps** : 30 secondes ⚡
